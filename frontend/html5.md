# HTML

```html
<!DOCTYPE HTML>
<!-- 表明是HTML文档，部分大小写 -->

<head>
<!-- 头部，前面的叫开始标签，后面的叫结束标签。通常标签都是成对的，右边带"/" -->
<!-- 包含所有的头部的标签元素，可以在此插入脚本，样式文件，以及各种 meta 信息 -->
    
    <meta charset="utf-8">
    <!-- 编码格式：UTF-8。charset：字符集 -->
    
    <title>标题</title>
    <!-- 不可缺少，1、浏览器工具栏的标题。2、收藏夹中的标题。3、搜索引擎结果界面的标题 -->
    
    <base herf="https://www.baidu.com" target="_blank">
</head>


<>
```

### maven 项目说明

##### maven 项目目录结构

```tex
/ (项目名)
|
|--pom.xml (项目说明文件 maven依赖)
|--src
|    |
|    |--main
|         |
|         |--test (测试代码)
|         |
|         |--java (存放java源代码)
|         |
|         |--webapp
|                |
|                |--WEB_INF
|                |      |
|                |      |--web.xml ()
|                |      |
|                |      |--classes (java编译后的class文件)
|                |
|                |--index.html (html/jsp文件)
|
|--target (项目构建输出文件)
```
